Smart Sales Analytics & Forecasting Dashboard

An end-to-end sales analytics web application that analyzes historical sales data and predicts future revenue using machine learning.

Tech Stack 
- Python
- Streamlit
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- HTML & CSS

Key Features
- Interactive KPI dashboard (Revenue, Profit, Units Sold)
- Sales trend visualization
- Machine learning-based revenue forecasting
- Clean, enterprise-style UI

 How to Run
```bash
pip install -r requirements.txt
python -m streamlit run app.py
